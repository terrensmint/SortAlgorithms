# Лабораторная №4 - Сортировки.

## Задача
### Требуется реализовать алгоритмы сортировки:
- Сортировка пузырьком
- Сортировка выбором
- Быстрая сортировка

### Протестировать сортировку на массивах размером:
- 10 элементов
- 1 000 элементов
- 10 000 элементов
- 100 000 элементов

Массивы генерируются случайным образом.

### Замерить и сравнить время работы сортировок. 
Вывести результат в виде таблицы.

### Дополнительно.
Для большего сравнения, программа выводит вторую таблицу с количеством перестановок элементов массива в разных алгоритмах.

## Установка и запуск
```bash
# Клонировать репозиторий
git clone https://github.com/terrensmint/SortAlgorithms.git
# Перейти в папку проекта
cd SortAlgorithms

# Компиляция
gcc -o program main.c arrays.c sorting.c

# Запуск программы
./program
```
